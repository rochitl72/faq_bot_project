
# FAQ Chatbot

A simple FAQ chatbot that retrieves answers from a predefined dataset.

## Features

- Loads FAQ data from a CSV file.
- Uses TF-IDF & cosine similarity for retrieving answers.
- Provides a Flask API for chatbot responses.
